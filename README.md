# 使用說明（給不會寫程式的你）

## 資料夾結構
```
site/
├── index.html       ← 網頁內容（你平常不太需要改這個）
├── css/style.css    ← 顏色、版面、動畫樣式
├── js/main.js       ← 互動功能（捲動進度條、頁籤切換、翻卡片）
├── images/          ← 把圖片放這裡（目前是空的，見下方）
└── README.md        ← 這份說明
```

## 你現在唯一要做的事：補上兩張圖片

`index.html` 裡面用到了這兩張圖：
- `brain-normal.jpg`
- `brain-adhd.jpg`

請把這兩張圖片檔案，放進 `images/` 資料夾裡，**檔名要完全一樣**（包含大小寫）。

放好之後，回來告訴我「圖片放好了」，我會幫你把 `index.html` 裡的路徑也一起檢查、修正成 `images/brain-normal.jpg` 這種正確路徑（目前路徑還沒指到 images 資料夾，我會幫你調整）。

## 之後怎麼部署上網（讓別人能用網址看到）

等內容都確認沒問題後，我會依你選擇的方式（GitHub Pages 或 Netlify），一步一步帶你操作，包含：
1. 註冊帳號（如果還沒有）
2. 上傳整個 `site/` 資料夾
3. 拿到網址、開啟確認顯示正常

你完全不需要打指令，跟著我給的步驟點滑鼠、複製貼上即可。
